## 🎯 Что вы получите:

После настройки сможете публиковать одной командой:

```bash
# Опубликовать статью
npm run fb:post insight ki-im-mittelstand-wo-sie-wirklich-hilft

# Опубликовать проект
npm run fb:post project kraftwerk-digital
```

---

## 📋 Что нужно для начала:

1. ✅ **Facebook Page** (страница, не личный профиль)
2. ⏱️ **10-15 минут** свободного времени
3. 🔑 **Доступ к Facebook Developers** (бесплатно)

---

## 🧪 Тестирование

После настройки проверьте подключение:

```bash
npm run fb:test
```

Эта команда проверит:
- ✅ Валидность токена
- ✅ Разрешения
- ✅ Доступ к странице
- ✅ API публикации
