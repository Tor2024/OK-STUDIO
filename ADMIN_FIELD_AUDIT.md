## 1. PROJEKTE (Projects)

### Структура данных (JSON):
```json
{
  "id": "kraftwerk-digital",
  "title": "KRAFTWERK DIGITAL",
  "category": "Web Relaunch",
  "description": "Kurzbeschreibung...",
  "fullDescription": "## Markdown Inhalt...",
  "image": "https://...",
  "completedAt": "MRZ 2025",
  "order": 1
}
```

### Поля в Admin панели:
| Поле | Тип | Обязательное | Соответствие | Примечание |
|------|-----|--------------|--------------|------------|
| `title` | text | ✅ Да | ✅ Да | Заголовок проекта |
| `category` | text | ✅ Да | ✅ Да | Категория (Web Relaunch, System Dev, AI & Automation) |
| `image` | text (URL) | ❌ Нет | ✅ Да | URL изображения |
| `completedAt` | text | ❌ Нет | ✅ Да | Дата завершения (текстовый формат: "MRZ 2025") |
| `description` | textarea | ❌ Нет | ✅ Да | Краткое описание для списка |
| `fullDescription` | textarea (Markdown) | ❌ Нет | ✅ Да | Полное описание с Markdown |
| `order` | number | ❌ Нет | ✅ Да | Порядок отображения |

### Особенности:
- ✅ Использует **отдельные JSON файлы** для каждого проекта
- ✅ Index файл **не содержит** `fullDescription` (оптимизация)
- ✅ При редактировании загружается **полный файл** с `fullDescription`
- ✅ При сохранении обновляются **оба файла** (индивидуальный + index)

## 🔄 WORKFLOW СООТВЕТСТВИЕ

### Фронтенд → Admin → GitHub → Фронтенд

1. **Фронтенд читает данные**:
   - Projects: `public/data/projects/index.json` (без fullDescription)
   - Insights: `public/data/insights/index.json` (без content)
   - Clients: `public/data/clients.json`
   - FAQs: `public/data/faqs.json`
   - Settings: `public/data/settings.json`

2. **Admin редактирует**:
   - Загружает полные данные из индивидуальных файлов (Projects, Insights)
   - Редактирует все поля
   - Сохраняет изменения

3. **GitHub получает commit**:
   - Обновляются индивидуальные файлы (Projects, Insights)
   - Обновляются index файлы (Projects, Insights)
   - Обновляются единые файлы (Clients, FAQs, Settings)

4. **Автоматический redeploy**:
   - Vercel/Netlify получает webhook
   - Пересобирает сайт
   - Новые данные отображаются на фронтенде
