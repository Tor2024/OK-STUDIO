## 🔒 Безопасность

### Реализованные меры:

1. **Хеширование**
   - SHA-256 алгоритм
   - Пароль никогда не хранится в открытом виде
   - Только хеш в коде

2. **Rate Limiting**
   - Максимум 5 попыток
   - Автоматическая блокировка
   - Счетчик попыток

3. **Session Management**
   - sessionStorage (очищается при закрытии браузера)
   - Автоматический logout
   - Безопасное хранение

4. **Валидация**
   - Проверка длины пароля
   - Проверка хеша
   - Защита от пустых значений

## 📝 Использование

### Для пользователя:

1. **Первый запуск:**
   ```bash
   npm run password:generate МойПароль123
   # Замени хеш в src/pages/Admin.tsx
   ```

2. **Вход:**
   - Открой `/admin`
   - Введи пароль
   - Введи GitHub Token
   - Работай с админ-панелью

3. **Выход:**
   - Нажми кнопку Logout
   - Очищается пароль и GitHub config

### Для разработчика:

```typescript
// Структура в Admin.tsx:

// 1. Константа с хешем
const ADMIN_PASSWORD_HASH = '...';

// 2. Функция хеширования
function hashPassword(password: string): Promise<string>

// 3. Функции управления auth
function getPasswordAuth(): boolean
function setPasswordAuth(value: boolean)

// 4. Компонент экрана входа
function PasswordScreen({ onAuth })

// 5. Проверка в главном компоненте
if (!isAuthenticated) {
  return <PasswordScreen onAuth={() => setIsAuthenticated(true)} />;
}
```

## ⚠️ ВАЖНО

### Перед использованием:

1. **ОБЯЗАТЕЛЬНО измени пароль по умолчанию!**
   - Текущий: `password`
   - Это небезопасно!

2. **Используй сильный пароль:**
   - Минимум 12 символов
   - Заглавные и строчные буквы
   - Цифры и спецсимволы

3. **Храни пароль в безопасном месте:**
   - Менеджер паролей (1Password, Bitwarden)
   - Не в Git!
   - Не в открытом виде!
