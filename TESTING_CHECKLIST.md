# Admin Panel - Полный тестовый план

## 🎯 ЦЕЛЬ ТЕСТИРОВАНИЯ

Убедиться, что Admin панель полностью функциональна и все поля работают корректно для управления контентом сайта.

## 🔐 1. АУТЕНТИФИКАЦИЯ

### 1.1 Passwort-Screen (Уровень 1)
- [ ] Открыть `/admin` - должен показаться экран с паролем
- [ ] Ввести неправильный пароль - должна появиться ошибка
- [ ] Проверить счетчик попыток (5 квадратиков внизу)
- [ ] После 5 неудачных попыток - должна появиться блокировка
- [ ] Ввести правильный пароль (`password`) - должен пройти на GitHub Token экран
- [ ] Проверить, что session сохраняется (обновить страницу - не должен запрашивать пароль снова)

### 1.2 GitHub Token (Уровень 2)
- [ ] Ввести неправильный token - должна появиться ошибка
- [ ] Ввести правильный token с правильными owner/repo/branch
- [ ] Проверить, что config сохраняется в localStorage
- [ ] Должен открыться Dashboard

### 1.3 Logout
- [ ] Нажать кнопку Logout (красная иконка справа вверху)
- [ ] Должен очиститься и пароль, и GitHub token
- [ ] Должен вернуться на Passwort-Screen
