## ⚙️ Настройка

### 1. Получи Gemini API Key

1. Зайди на [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Нажми **"Get API Key"** → **"Create API key"**
3. Скопируй ключ (начинается с `AIza...`)

### 2. Добавь в Vercel Environment Variables

Зайди в **Vercel Dashboard** → твой проект → **Settings** → **Environment Variables**

Добавь:
```
GEMINI_API_KEY=AIzaSy...твой_ключ
VITE_GEMINI_API_KEY=AIzaSy...твой_ключ
```

💡 **Можно добавить несколько ключей** через запятую для резерва:
```
GEMINI_API_KEY=AIzaSy...ключ1,AIzaSy...ключ2,AIzaSy...ключ3
```

### 3. Включи Telegram уведомления (опционально)

Чтобы получать сообщения о заинтересованных клиентах:

```
TELEGRAMM_TOKEN=8513318740:AAHKFRm...
TELEGRAMM_ID=6229634947
```

У тебя уже настроено! ✅

## 🆘 Помощь

### Бот не отвечает?
1. Проверь Gemini API Key в Vercel
2. Открой Console (F12) → смотри ошибки
3. Проверь что бот включён в админке

### Бот отвечает невпопад?
1. Отредактируй `SYSTEM_PROMPT` в `api/chatbot.ts`
2. Сделай инструкции более конкретными
3. Добавь больше примеров диалогов

### Не приходят уведомления в Telegram?
Проверь `TELEGRAMM_TOKEN` и `TELEGRAMM_ID` в Vercel
