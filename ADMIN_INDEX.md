## ⚠️ ВАЖНО ПЕРЕД PRODUCTION

### ОБЯЗАТЕЛЬНО:
1. **Сменить пароль!**
   ```bash
   npm run password:generate
   ```
   Заменить хеш в `src/pages/Admin.tsx` (строка 32)

2. **Создать GitHub Token**
   - GitHub → Settings → Developer settings → Personal access tokens
   - Scope: `repo` (full control)

3. **Протестировать**
   - Пройти чеклист: [TESTING_CHECKLIST.md](TESTING_CHECKLIST.md)
   - Проверить на разных устройствах
   - Проверить GitHub интеграцию
