## ✅ ПОЛНАЯ ИНТЕГРАЦИЯ С ADMIN PANEL

### 1. **Чтение данных (READ)**
✅ **Работает полностью**
- Страница `/approach` использует `useCollection<any>('faqs')`
- Загружает данные из `public/data/faqs.json`
- Отображает все вопросы и ответы в аккордеоне
- Автоматически обновляется при изменении данных

### 2. **Создание (CREATE)**
✅ **Работает полностью**
- В Admin Panel → Tab "FAQ"
- Форма "Neue FAQ" слева
- Поля:
  - **Frage** (Вопрос) — обязательное
  - **Antwort** (Ответ) — обязательное
  - **Reihenfolge** (Порядок) — опциональное
- Кнопка "SPEICHERN & DEPLOYEN"
- Сохраняет в `faqs.json` → Commit → Deploy

### 3. **Редактирование (UPDATE)**
✅ **Работает полностью**
- Синяя кнопка "Edit" (карандаш) на каждом FAQ
- Загружает данные в форму слева
- Заголовок меняется на "FAQ bearbeiten"
- Кнопка "AKTUALISIEREN" для сохранения
- Кнопка "ABBRECHEN" для отмены
- Сохраняет изменения → Commit → Deploy

### 4. **Удаление (DELETE)**
✅ **Работает полностью**
- Красная кнопка "Delete" (корзина) на каждом FAQ
- Подтверждение: "FAQ löschen?"
- Удаляет из `faqs.json` → Commit → Deploy
