# Demo Mode - Просмотр Admin Panel без GitHub

## 🎯 Что это?

**Demo Mode** позволяет просматривать Admin Panel и все его функции **без подключения к GitHub**.

Это полезно для:
- 👀 Просмотра интерфейса
- 🎨 Проверки дизайна
- 📱 Тестирования responsive
- 🧪 Демонстрации клиентам

---

## 🚀 Как использовать

### Шаг 1: Запустить dev сервер

```bash
npm run dev
```

### Шаг 2: Открыть Admin Panel

```
http://localhost:5173/admin
```

### Шаг 3: Ввести пароль

```
Пароль: password
```

### Шаг 4: Нажать "DEMO MODUS"

На экране GitHub Token нажмите кнопку:

```
┌─────────────────────────────────┐
│ [⚡] DEMO MODUS (Nur Ansicht)   │
└─────────────────────────────────┘
```

### Шаг 5: Готово!

Теперь вы видите Admin Panel с локальными данными из `public/data/`

## 📊 Откуда берутся данные

Demo Mode загружает данные из локальных файлов:

```
public/data/
├── projects/index.json    ← Проекты
├── insights/index.json    ← Статьи
├── clients.json           ← Клиенты
├── faqs.json             ← FAQ
└── settings.json         ← Настройки
```

**Это те же файлы, что использует фронтенд!**

2. Ввести пароль снова
3. Ввести GitHub Token
4. Нажать **VERBINDEN**

### Из GitHub Mode в Demo Mode:
1. Нажать **Logout**
2. Ввести пароль снова
3. Нажать **DEMO MODUS**

## 🐛 Troubleshooting

### Проблема: "Cannot read properties of undefined"

**Решение:** Убедитесь, что файлы существуют в `public/data/`

```bash
# Проверить файлы
ls public/data/
```

### Проблема: Пустые данные

**Решение:** Проверьте формат JSON файлов

```bash
# Проверить JSON
cat public/data/projects/index.json
```

### Проблема: Не загружаются изображения

**Решение:** Это нормально в Demo Mode. Изображения загружаются с внешних URL.
