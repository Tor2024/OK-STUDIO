## 🚀 Быстрая настройка (2 минуты)

### Шаг 1: Сгенерируй хеш

```bash
npm run password:generate МойНовыйПароль123
```

**Пример вывода:**
```
✅ Хеш успешно сгенерирован!

Пароль: МойНовыйПароль123
Хеш:    a1b2c3d4e5f6...
```

### Шаг 2: Скопируй хеш

Скопируй длинную строку (хеш) из вывода команды.

### Шаг 3: Замени в коде

1. Открой файл: `src/pages/Admin.tsx`

2. Найди строку (примерно строка 20):
   ```typescript
   const ADMIN_PASSWORD_HASH = '5e884898da28047151d0e56f8dc6292773603d0d6aabbdd62a11ef721d1542d8';
   ```

3. Замени хеш на свой:
   ```typescript
   const ADMIN_PASSWORD_HASH = 'a1b2c3d4e5f6...'; // Твой хеш
   ```

4. Сохрани файл

### Шаг 4: Готово!

Теперь используй свой новый пароль для входа в админ-панель.

---

## 🔑 Вход в админ-панель

1. Открой: `http://localhost:3000/admin` (или твой домен)

2. **Экран 1: Пароль**
   - Введи свой новый пароль
   - Нажми "ZUGANG GEWÄHREN"

3. **Экран 2: GitHub**
   - Введи GitHub Token
   - Введи Owner, Repo, Branch
   - Нажми "VERBINDEN"

4. **Готово!** Ты в админ-панели

---

## 💡 Требования к паролю

### Хороший пароль:
- ✅ Минимум 12 символов
- ✅ Заглавные и строчные буквы
- ✅ Цифры
- ✅ Специальные символы (!@#$%^&*)

### Примеры:
- `MySecure2024Pass!`
- `OKstudio#Admin99`
- `Kreuztal@Web2024`

### ❌ Плохие пароли:
- `password`
- `123456`
- `admin`

---

## 🛡️ Безопасность

### Что защищено:
- ✅ Пароль хешируется (SHA-256)
- ✅ Никогда не хранится в открытом виде
- ✅ Rate limiting (5 попыток)
- ✅ Автоматическая блокировка
- ✅ Session-based аутентификация

### Рекомендации:
1. Используй сильный пароль
2. Не делись паролем
3. Меняй пароль каждые 3-6 месяцев
4. Храни в менеджере паролей (1Password, Bitwarden)

---

## 🆘 Проблемы?

### Забыл пароль?

Просто сгенерируй новый хеш и замени в коде:
```bash
npm run password:generate НовыйПароль123
```

### Слишком много попыток?

Подожди 5 минут или очисти sessionStorage в DevTools (F12).

### Пароль не работает?

Проверь, что хеш правильно скопирован без лишних пробелов.
