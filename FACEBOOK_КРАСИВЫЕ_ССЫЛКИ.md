## 📋 Что нужно сделать:

### Шаг 1: Закоммитьте изображения
```bash
git add public/og
git commit -m "Add OG images for social media"
git push
```

### Шаг 2: Дождитесь деплоя на Vercel
Vercel автоматически задеплоит изменения (1-2 минуты).

### Шаг 3: Проверьте превью
Откройте Facebook Sharing Debugger:
```
https://developers.facebook.com/tools/debug/
```

Вставьте ссылку:
```
https://ok-studio-umber.vercel.app/insights/ki-im-mittelstand-wo-sie-wirklich-hilft
```

Нажмите **"Debug"** - увидите превью!

### Шаг 4: Опубликуйте пост
```bash
npm run fb:post insight ki-im-mittelstand-wo-sie-wirklich-hilft
```

### Шаг 5: Проверьте на Facebook
Откройте вашу страницу - должна быть красивая карточка!

## 🐛 Если изображение не появляется:

### Проблема: Facebook показывает старое превью
**Решение:** Очистите кеш Facebook

1. Откройте: https://developers.facebook.com/tools/debug/
2. Вставьте ссылку на статью
3. Нажмите **"Scrape Again"** (Обновить)
4. Facebook обновит превью

### Проблема: Изображение не загружается
**Решение:** Проверьте, что файл доступен

Откройте в браузере:
```
https://ok-studio-umber.vercel.app/og/insights/ki-im-mittelstand-wo-sie-wirklich-hilft.svg
```

Если файл не открывается - значит не задеплоился. Проверьте Vercel.

### Проблема: Facebook не видит изображение
**Решение:** SVG не всегда поддерживается

Facebook лучше работает с JPG/PNG. Можно:

**Вариант A:** Конвертировать SVG в PNG онлайн
1. Откройте: https://cloudconvert.com/svg-to-png
2. Загрузите SVG из `public/og/`
3. Скачайте PNG
4. Замените SVG на PNG

**Вариант B:** Использовать существующие изображения
Если у вас есть изображения проектов в `public/images/`, можно использовать их.
