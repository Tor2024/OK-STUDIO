# FAQ — Быстрая Инструкция 🇷🇺

**Где находится раздел FAQ на сайте?**  
👉 Страница **`/approach`** (Methodik) — внизу, после описания 4-фазного процесса

---

## 📍 КАК НАЙТИ FAQ НА САЙТЕ

1. Открыть сайт
2. Перейти в меню → **"Methodik"** (или `/approach`)
3. Прокрутить вниз до конца страницы
4. Увидите секцию **"HÄUFIG_GESTELLTE_FRAGEN (FAQ)"**

**Внешний вид:**
- Аккордеон (раскрывающиеся вопросы)
- Клик на вопрос → открывается ответ
- Клик еще раз → закрывается

---

## ✏️ КАК УПРАВЛЯТЬ FAQ ЧЕРЕЗ ADMIN PANEL

### 1️⃣ ДОБАВИТЬ НОВЫЙ ВОПРОС

```
Admin Panel → Tab "FAQ" → Форма слева:

┌─────────────────────┐
│ Neue FAQ            │
├─────────────────────┤
│ Frage:              │
│ [Ваш вопрос?]       │
│                     │
│ Antwort:            │
│ [Ваш ответ...]      │
│                     │
│ Reihenfolge:        │
│ [4]                 │
│                     │
│ [SPEICHERN & DEPLOY]│
└─────────────────────┘
```

**Шаги:**
1. Заполнить поле **"Frage"** (вопрос)
2. Заполнить поле **"Antwort"** (ответ)
3. Указать **"Reihenfolge"** (порядок: 1, 2, 3...)
4. Нажать **"SPEICHERN & DEPLOYEN"**
5. ✅ Готово! FAQ появится на сайте через 2-3 минуты

---

### 2️⃣ РЕДАКТИРОВАТЬ ВОПРОС

```
Список FAQ справа:

┌─ FAQ 1 ──────────────── ✏️ 🗑️ ─┐
│ Wie lange dauert...?           │
│ Ein Standard-Relaunch...       │
└────────────────────────────────┘
         ↑
    Клик на ✏️
```

**Шаги:**
1. Найти нужный FAQ в списке справа
2. Нажать **синюю кнопку** ✏️ (Edit)
3. Данные загрузятся в форму слева
4. Изменить текст
5. Нажать **"AKTUALISIEREN"**
6. ✅ Изменения сохранены!

---

### 3️⃣ УДАЛИТЬ ВОПРОС

```
Список FAQ справа:

┌─ FAQ 2 ──────────────── ✏️ 🗑️ ─┐
│ Was kostet...?                 │
│ Unsere Projekte...             │
└────────────────────────────────┘
                            ↑
                       Клик на 🗑️
```

**Шаги:**
1. Найти FAQ для удаления
2. Нажать **красную кнопку** 🗑️ (Delete)
3. Подтвердить: **"FAQ löschen?"** → OK
4. ✅ FAQ удален!
